GithubToDoxygen
=========

Read current commit from Github and generate a doxygen-documentation from it

**Licence:** GNU GPLv3

**work status:** pre Alpha, still in Developing

####How to use

1. Define Git Clone-Url in config.cfg
2. Run Programm:
```
./GithubToDoxygen.sh
```
